Types of List

  1. Ordered listing ( ol tags )
    Following are the types of li tag
    1. Default type of list indexing is numbers
    2. type = "a" : Starts the list indexing with small letters
    3. type = "A" : Gives indexing with capital letters
    4. type = "i" : Gives indexing with roman numbers
    5. type = "I" :Gives capital roman numbers Indexing
  • Unordered List (ul tag)
    Following are the types of li tag
    • Default type of indexing ( type="circle" )
    • type="square"
    • type="disc"
    • type = "none"
